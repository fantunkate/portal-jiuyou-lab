# portal-jiuyou-lab

本仓库用于归档和发布多个独立的 HTML 页面，内容涵盖各类静态网页资源，不针对特定域名或网站。

## 项目简介

portal-jiuyou-lab 是一个静态页面归档仓库，主要用于存放和发布独立的 HTML 文件。这些页面可被直接访问或嵌入其他环境中，适用于展示、测试、存档等用途。

## 目录说明

```
├── pages/          # 存放各独立 HTML 页面
├── assets/         # 页面所需的静态资源（CSS、JS、图片等）
├── index.html      # 仓库主页或导航页（可选）
└── README.md       # 本文件
```

- `pages/` 目录下每个 HTML 文件为一个独立页面，命名建议反映页面内容或功能。
- `assets/` 目录用于存放页面引用的资源文件，可按需组织子目录。

## 页面归档说明

- 所有页面均为独立 HTML 文件，不依赖外部数据库或后端服务。
- 页面可包含样式、脚本、多媒体等资源，建议将资源文件置于 `assets/` 目录下。
- 添加新页面时，请确保文件路径正确，避免资源引用错误。
- 本仓库不承诺页面的长期可用性，但会尽力维护文件完整性。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面，或修正现有页面问题。
- 提交前请确保页面内容合规，不包含恶意代码、侵犯版权或违反平台政策的内容。
- 仓库维护者保留审核和合并的权利。
